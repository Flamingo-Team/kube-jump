任务列表
=========

这里介绍任务列表的相关的功能。

.. contents:: Topics

.. _view_asset_works:

查看任务列表
````````````

任务列表页面默认展示一周内所有任务。点击标题可根据当前字段进行排序。

.. _invoke_asset_work:

手动执行任务
````````````

在任务列表中找到要手动执行的任务，点击“动作”标题下的“执行”按钮，完成手动执行当前任务。

.. _delete_asset_work:

删除任务
`````````

在任务列表中找到要删除的任务，点击“动作”标题下的“删除”按钮，完成删除当前任务。

.. _detial_asset_work:

查看任务详情
`````````````

在任务列表中找到要查看的任务，点击要查看的任务名称，即可进入任务详情页面。
