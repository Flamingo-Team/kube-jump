标签管理
============

这里介绍标签管理的功能。

.. contents:: Topics

.. _view_label_list:

查看标签列表
````````````````

点击页面左边“资产管理”菜单下的“标签管理”按钮，进入标签列表页面，产看标签的名称、值、资产数等信息。

.. _create_label:

创建标签
````````````

点击页面左上角“创建标签”按钮，进入创建标签页面，填写标签的名称、值等信息，选择资产，点击“提交”按钮，标签创建完成。

.. _update_label:

更新标签
````````````

点击页面右边动作栏的“更新”按钮，进入更新标签页面，编辑标签信息，点击“提交”按钮，标签更新完成。

.. _delete_label:

删除标签
`````````

点击页面右边动作栏的“删除”按钮，弹出删除确认框，点击“确认”按钮，完成标签删除。
